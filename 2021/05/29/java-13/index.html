三、Soket（一）

  1. 套接字（Socket）开发网络应用程序被广泛采用
  2. 通信的两端都有Socket，是两台机器间通信的端点
  3. 网络通信其实就是Socket间的通信
  4. Socket允许把网络连接当成一个流，数据在两个Socket之间通过IO传输
  5. 一般吧主动发起通信的应用程序称为客户端，等待通信请求的为服务端
