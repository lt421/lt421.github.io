二、预处理命令和宏定义

2.1、预处理命令基本介绍

  1. 以#开头的命令称为预处理
  2. 编译之前对源文件进行简单加工的过程
  3. 预处理是C语言的一个重要功能，由预处理程序完成，当一个源文件进行编译时，系统自动调用预处理程序对源程序中的预处理部分作处理，处理后自动进行编译
  4. C语言提供了多种预处理功能，如宏定义、文件包含，条件编译…

开发一个C语言程序，让它暂停五秒后再输出“cyfwlp”，并且要求跨平台，在Windows和linux下都能运行

提示：

  1. Windows 暂停函数的原型 void Slepp(DWORD dwMillsecounds)，参数的单位是毫秒，位于<windows.h>头文件
  2. linux 暂停函数 unsigned int sleep (unsigned int secound)，参数的单位值秒位于<unistd.h>
  3. #if #elif #endif 都是预处理命令，它们都是在编译前由预处理程序来执行的

2.2、宏定义

#define叫做宏定义命令，用一个标识符标识一个字符串，源文件出现标识符进行简单的替换

宏定义的形式：

  1. 宏名命名与变量相同；字符串可以是数字，表法式，if语句，函数等
  2. 程序中反复使用的表达式就可以使用宏定义

Notes

  1. 预处理程序对表达式不作任何检查，若有错误，只能在编译已被宏展开的的源程序时发现
  2. 宏定义不需要加；（若加了会把;也一起替换）
  3. 作用域为源程序结束，终止可以使用#undef命令
  4. 习惯使用大写（也可以使用小写）
  5. 可以使用宏定义表示数据类型，书写方便
    #define UINT unsigned int
  6. 与typedef区别：宏定义只是简单的字符串代替，而typedef是在编译阶段由编译器处理的，给原有的数据类型起新的名字，作为一种新的数据类型

2.3、带参数的宏定义

形式参数之间可以出现空格，但是宏名与形式参数之间不能有空格

2.4、C语言预处理命令总结

指令 说明
# 空指令，没有任何效果
#include 引入头文件
#define 宏定义
#undef 取消宏定义
#if 如果条件为真，则编译
#ifdef 如果宏已经定义，则编译
#ifndef 如果宏没有定义，则编译
#elif else if 为真则编译
#endif 结束 if

3.2、数组定义和内存布局

定义和初始化方式：

  1. 数组名就代表数组的首地址，即a[0]地址
  2. 数组的各个元素在内存中是连续分布的

Notes

  1. 数组一旦定义/声明，长度是固定的，不能动态变化
  2. 数组若没有赋值：
    全局数组默认0
    非全局数组是机器辣鸡值（系统原来分配给这块内存的值）

3.4、字符数组与字符串

用来存放字符的数组称为字符数组，C语言中没有string类型的变量，因此 C语言中 字符数组 == 字符串

Notes

  1. 在C语言中，字符串实际上是使用null字符（’\0’）终止的一维字符数组。因此，一个以null结尾的字符串，包含了组成字符串的字符

  2. ‘\0’是ASCII码表中第0个字符，用null表示，称为空字符。该字符既不能显示，也不是控制字符，输出该字符不会有任何效果，它在C语言中仅作为字符串结束标致

  3. 字符串在内存的布局分析

  4. 如果在给某个字符数组赋初值时，赋给的元素个数小于该数组的长度，会自动加’\0’表示字符串结束

    赋给元素的个数等于数组的长度，则不会自动添加’\0’

指针指向字符数组

Notes:

  1. 如果定义了一个字符数组，那么它有确定的内存地址（即字符数组是一个常量）；而定义了一个字符指针变量，它并未指向某个确定的字符数据，并且可以多次赋值
  2. printf(“%s”,str)可以输出str指向的字符串

3.5、字符串相关函数

序号 函数 目的
1 strcpy(s1,s2) 复制s2到s1
2 strcat(s1,s2) 连接s1,s2
3 strlen(s1) 返回s1长度
4 strcmp(s1,s2) 如果s1和s2相同，返回0，s1<s2返回小于0，s1>s2返回大于0
5 strchr(s1,ch) 返回一个指针，指向字符串s1中字符ch第一次出现的位置
6 strstr(s1,s2) 返回一个指针，指向字符串s1中字符串s2的第一次出现的位置

Notes:

  1. ‘\0’用于检测和判断字符串是否结束，字符串长度不会统计’\0’
  2. 系统对字符串常量自动加’\0’表示结束，如”cyfwlp”共有6个字符，但是在内存中占7个字节
  3. 如果定义的字符个数比数组长度小，系统会将剩余的元素空间全部填充’\0’
