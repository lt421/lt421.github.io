考试前花几天速成oracle

1、PL/SQL

pl/sql编程语言是对sql语言的拓展，使得sql语言具有过程化编程的特性

主要用来编写存储过程和存储函数

2、游标

cursor ：存放多个对象

3、存储过程

存储过程：提前编写好的PL/SQL语句，供程序直接调用

4、存储函数

存储函数与存储过程的区别：

存储函数有返回值 -> 存储函数可以用来自定义函数

5、触发器

触发器：指定一个规则，在我们做增删改操作的时候，只要满足规则，自动触发，无需调用
语句级触发器
行级触发器 for each row
for each row 为了使用 ：old 和 ：new 对象或者一行记录

触发语句与伪记录变量的值：

触发语句 :old :new
insert null 将要插入的数据
update 更新前的数据 更新后的数据
delete 删除前的值 null
